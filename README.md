# Urban Seasonal Intelligence System - Project Summary

**Urban Seasonal Intelligence System** is an AI-powered web application that analyzes seasonal environmental impacts across Indian metro cities, providing data-driven insights for urban planning and management.

The application features a multi-page flow: a landing page showcasing capabilities, a city/season selection interface, an interactive dashboard with AI-generated insights (powered by Groq API), and a journey planner that analyzes environmental conditions between two locations.

Core functionality includes real-time visualization of AQI, temperature, humidity, rainfall, traffic patterns, heat stress, and flood risk data through interactive maps (Leaflet.js with heatmaps), charts (Chart.js), and district-wise vulnerability comparisons.

The tech stack is pure frontend (HTML/CSS/JavaScript) with Tailwind CSS for styling, featuring animated backgrounds, glassmorphic UI components, and dynamic data visualization without requiring a backend server.

Key features include seasonal data analysis for 8 major Indian cities (Mumbai, Delhi, Bangalore, etc.), AI-powered relationship insights between environmental factors, and actionable recommendations for infrastructure planning and resource management.
